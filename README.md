# React Web App

### Development
```bash
git clone https://github.com/aslamj/react-web-app.git
cd react-web-app
yarn
yarn start
```

Dev web server runs at [localhost:3000](http://localhost:3000)

### Testing
```bash
yarn jest:coverage
```
